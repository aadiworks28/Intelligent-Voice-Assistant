# Intelligent Voice Assistant (MVP)

This project is a beginner-friendly intelligent voice assistant built using Python.
It supports:
- Microphone audio recording
- Speech-to-text (ASR)
- Intent recognition
- Command execution
- Text-to-speech (TTS) replies
